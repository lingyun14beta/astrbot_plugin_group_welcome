#  AstrBot Group Welcome (入群欢迎插件)

**人机代码，勿喷qwq**

一个专为 AstrBot 打造的、高度智能的入群欢迎插件。支持 **普通/AI 混合欢迎**、**群专属欢迎语智能设置**、**私聊群规** 以及完善的 **黑白名单** 管理。

> **当前版本**: 1.0.9

## ⚠️ 依赖说明 
1. **🔌 平台/协议依赖**：**依赖 OneBot 11**。本插件绑定 `aiocqhttp` 适配器。**配合NapCat的框架使用**。（暂不支持其他平台）。
2. **💡 运行建议**：为确保事件监听注册成功，建议使用 **AstrBot v4.12.0** 或更高版本。若机器人启动后入群欢迎未即时触发，请尝试**重启机器人**或**重载插件**。

---

## ✨ 核心特性

*   **🎨 智能群专属设置**: 支持为不同群单独设置欢迎语。
*   **📱 智能状态面板**: 一条 `/welcome status` 指令，自动识别是在群聊还是私聊发送，智能展示当前群的具体欢迎语或全局统计信息。
*   **🛡️ 管理**: 
    *   提供【黑/白名单】管理机制。
    *   考虑到 QQ 会话风控，**【私聊群规】功能默认关闭**。

---

## 🛠️ 安装与配置

### 1. 安装
将插件文件夹（包含 `main.py`, `_conf_schema.json`, `metadata.yaml`）放入 AstrBot 的 `data/plugins/` 目录中，重启 AstrBot 即可生效。

### 2. 后台设置 (WebUI)
进入 AstrBot 后台管理系统，在插件配置页面可以设置以下功能：

| 配置项 | 说明 |
| :--- | :--- |
| **🎉 全局默认欢迎语** | 支持变量 `{name}` (昵称) 和 `{count_text}` (第几位成员) |
| **📊 开启群人数统计** | 是否在欢迎语中追加“你是当前群里第 X 位成员” |
| **📩 开启私聊群规** | 是否尝试私聊发送群规 (**默认关闭以防风控**) |
| **📜 群规内容** | 私聊发送的具体文字内容 |
| **🤖 开启 AI 个性化欢迎** | 是否追加 AI 祝福（**自动使用 AstrBot 全局默认大模型**） |
| **📝 AI 提示词 (Prompt)** | 指导 AI 生成欢迎语的预设系统词 |
| **⏱️ 防刷屏冷却 (秒)** | 同一人短时间重复入群的触发间隔 (默认 `300` 秒) |
| **✅ / 🚫 黑白名单** | 逗号分隔的群号列表 |

> ⚠️ **提示**：强烈建议不要在 WebUI 中手动修改 `[系统数据] 群专属配置`，请使用下方的 `/welcome set` 指令在 QQ 内直观配置以防 JSON 损坏。

## 💻 指令列表

所有指令需拥有 **管理员** 权限。指令前缀默认为 `/`，请根据你的 AstrBot 配置调整。

### 📝 欢迎语定制 (Set)
*指令支持直接输入包含空格的长句，系统会自动识别，无需添加引号！*

*   **在群聊中发送：**
    *   `/welcome set 欢迎 {name} 加入本群！`：设置**本群**专属欢迎语。
    *   `/welcome set reset`：清除本群专属设置，恢复全局默认。
*   **在私聊中发送：**
    *   `/welcome set <群号> 欢迎 {name} 加入本群！`：为**指定群**设置欢迎语。
    *   `/welcome set reset <群号>`：清除指定群设置。

### 📊 智能状态查询 (Status)
输入 `/welcome status` 智能获取运行状态：
*   **在群聊中发送**：展示基础开关 + **本群**的具体欢迎语及来源。
*   **在私聊中发送**：展示基础开关 + 全局已设置专属欢迎语的群数量。
*   **私聊带群号** (`/welcome status 123456`)：展示基础开关 + **该群**的具体欢迎语及来源。

### ⚙️ 快速开关控制
*   `/welcome count on` / `off` ➡️ 开/关 群人数统计
*   `/welcome rules on` / `off` ➡️ 开/关 私聊群规功能
*   `/welcome ai on` / `off` ➡️ 开/关 AI 个性化欢迎

### 🛡️ 黑白名单管理
名单逻辑：**优先判定白名单；白名单为空时才判定黑名单。**

*   **白名单**：`/welcome wl add <群号>` / `/welcome wl del <群号>` / `/welcome wl list`
*   **黑名单**：`/welcome bl add <群号>` / `/welcome bl del <群号>` / `/welcome bl list`

## 🧩 常见问题 (Q&A)

**Q: 为什么开启了私聊群规，但新人进群没有收到私聊？**
A: 这是正常的。新进群的成员可能在 QQ 隐私设置中关闭了“允许陌生人发起临时会话”，或者近期大环境风控严格，机器人的“临时会话”权限被腾讯短暂限制。**建议将重要的群规直接写在群欢迎语中。**

**Q: 为什么我想用 AI 欢迎，但它没有生效？**
A: 请确认：
1. 使用 `/welcome ai on` 开启了功能。
2. 你的 AstrBot 已经配置了全局有效的大模型（如 OpenAI、DeepSeek 等）。只要平时 @机器人 能正常对话，欢迎插件就能正常调用它生成内容。
